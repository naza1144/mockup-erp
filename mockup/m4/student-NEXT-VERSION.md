## ข้อมูลจาก module ต่าง ๆ
| ข้อมูล | แหล่ง | หมายเหตุ |
|--------|-------|----------|
| ชื่อ-นามสกุล, อีเมล, เบอร์โทร, รูป | M0 API | แก้ไขที่ M0 Account Center |
| รหัสนักศึกษา, หลักสูตร, ชั้นปี | M4 registry.student | import/API |
| GPA, หน่วยกิต | M4 (E3 — next version) | ต้องมี Grade Management ก่อน |
| อาจารย์ที่ปรึกษา | M3 API | student.advisor_id → faculty.instructor |
| หลักสูตร/รุ่นปี | M1 API | program_version_id → curriculum |
